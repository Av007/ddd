# NestJs DDD


DDD in nest.js

## Run

```sh
docker compose up
```

In browser

```sh
http://localhost:3000/docs
```

Steps

* Try first call /api/init/{name} with value `harry+potter`
* Try second call and see results with filter
* try third call to see logs range in format `2025-01-22`
